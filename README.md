# Rex

A coding language beating HTML, CSS, and JS, and simple as English.

## Overview

Rex is a modern programming language designed to replace the complexity of HTML, CSS, and JavaScript with a simple, English-like syntax. Write web applications in a fraction of the code.

## Installation

```bash
cargo install rex
```

## Quick Start

```rex
make title = "Hello World"
make greeting = "Welcome to Rex"

show greeting
```
